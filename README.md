# Automated-Disk-Sanitiser
The Duplicate File Remover is a python utility that scans a directory, detects duplicates files using MD5 checksum comparisons, and deletes extra copies while keeping one original file.
This tool helps in cleaning storage and removing redundant files automatically.
Features - 
Recursive directory scanning
Detects duplicates using hashing (MD5)
Shows duplicate file groups
Deletes extra copies safely
Keeps one original file
Works with all file types
